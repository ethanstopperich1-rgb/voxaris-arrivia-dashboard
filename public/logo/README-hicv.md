# HICV logo asset slot

Drop the official Holiday Inn Club Vacations logo here as `hicv.svg`
(or `hicv.png` — whatever the brand team approves). The cassie-demo
page references `/logo/hicv.svg` and will pick up the file automatically
on next deploy.

Until then, the page renders the wordmark as styled text so the demo
isn't blocked by missing artwork.

Recommended: SVG, white-on-green variant for the navy header. Aspect
ratio close to 200×56 to match the Arrivia logo placement.
